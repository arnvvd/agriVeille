# Agriveille
> Funny website to present a monitoring about connected agriculture . 

![](thumbnail.png)


## Installation

### Install Dependencies
```sh
npm install
```

### Run Server on localhost:8080
``` bash
npm run dev
```

### Build your app
``` bash
npm run build
```
